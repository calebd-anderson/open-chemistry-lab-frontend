"The Open ChemLab" project was started as a Bachelor of Science capstone project. The primary use case is educational through discovering real-world facts about chemistry by experimenting with different combinations of elements from the periodic table of elements. An external RESTful API data services named PubChem is used.

Created using service oriented architecture. The source code can be found on GitHub here:

  • Backend - https://github.com/calebd-anderson/open-chemistry-lab-backend
  • Frontend - https://github.com/calebd-anderson/open-chemistry-lab-frontend

Made up of the follwing technologies:

  • Spring Boot - Web API
  • MongoDB - NoSQL persistence
  • Angular - Frontend
  • GSAP - Animations
  • Periodic Table - periodic table CSS
